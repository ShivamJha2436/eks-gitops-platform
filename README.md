# eks-gitops-platform

A production-style Kubernetes GitOps platform built locally using Kind and ArgoCD.

## Stack
- Kubernetes (Kind)
- GitOps (ArgoCD)
- Helm-based deployments
- Monitoring (Prometheus + Grafana)
- Autoscaling manifests
- AWS-ready Terraform infrastructure (not applied)

## Goal
Simulate a real EKS GitOps workflow with zero cloud cost.
